# Resume Screening and Classification using DistilBERT

## Description
AI-powered resume screener that automatically classifies resumes into job categories, extracts skills, detects experience level, and provides a match score using **DistilBERT**. Helps recruiters quickly identify suitable candidates.

## Features
- Resume classification into predefined job roles
- Skill extraction and missing skills identification
- Experience level detection (Entry, Mid, Senior)
- Resume match score with visual donut chart
- Supports PDF, DOCX, and TXT resume formats

## Tech Stack
- Python
- PyTorch
- Hugging Face Transformers (DistilBERT)
- Streamlit (Web UI)
- Pandas, NumPy, Scikit-learn
